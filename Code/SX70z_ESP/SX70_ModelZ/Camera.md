# Camera 工作模式与曝光流程

## 菜单模式

| `menu` | 模式 | `cam_mode` 显示 | `shut_mode` | 快门速度来源 |
|--------|------|----------------|-------------|-------------|
| 0 | AUTO | `"AUTO"` | `'1'` | 测光自动计算 (`auto_shutter_pos`) |
| 1 | BULB (B) | `"B"` | `'B'` | 忽略，S1T 释放结束曝光 |
| 2 | TIME (T) | `"T"` | `'T'` | 忽略，S1T 按两次结束曝光 |
| 3 | MANUAL (M) | 快门速度字符串 | `'1'` | 用户手动选择（上/下键） |
| 10 | SELF-TIMER | `"---"` | 不变 | 禁止拍摄（shutter_task 跳过） |

## 全局参数

```c
typedef struct {
    bool if_display;            // OLED 是否可用
    bool has_flash;             // 闪光灯是否接入（control_task 防抖检测）

    metering_state_t metering;  // 测光
    button_state_t button;      // 按键

    uint8_t menu;               // 菜单层级
    char cam_mode[8];           // 当前模式显示字符串
    char shut_mode;             // 快门模式：'1'=NORMAL, 'B'=BULB, 'T'=TIME
    uint8_t shutter_speed;      // 快门速度索引 (0-26)

    bool test_led_level;        // LED 测试电平
} camera_state_t;
```

### camera_state.shut_mode 语义变化

旧代码中 `shut_mode` 混合表示闪光/模式两种状态：

- `'0'` = 闪光灯曝光路径
- `'1'` = 普通曝光
- `'B'` = B 门
- `'T'` = T 门

新代码中 `shut_mode` **仅表示用户选择的拍摄模式**，闪光灯作为独立 `has_flash` 布尔值传递：

- `'1'` = NORMAL 曝光（AUTO 和 MANUAL 共用）
- `'B'` = B 门
- `'T'` = T 门

`shutter_task` 内通过 `use_flash = camera_state.has_flash` 判断是否执行闪光相关步骤。

## 行为矩阵

| 模式 | 无闪光灯 | 有闪光灯 |
|------|---------|---------|
| **AUTO** | 测光 → `auto_shutter_pos` → `mode='1'` 普通曝光 | 快门固定 index 13 (1/30s) → `mode='1'` + `use_flash` → 闪光曝光 |
| **BULB** | `mode='B'` → 开快门 → 15ms → 等 S1T 释放 → 关快门 | `mode='B'` + `use_flash` → 开快门 → 15ms+32ms → **触发闪光** → 等 S1T 释放 → 关快门 |
| **TIME** | `mode='T'` → 开快门 → 等 S1T 释放 → 等再次按下 → 关快门 | `mode='T'` + `use_flash` → 开快门 → 47ms → **触发闪光** → 等 S1T 释放 → 等再次按下 → 关快门 |
| **MANUAL** | `mode='1'` → 用户指定快门速度 → 普通曝光 | `mode='1'` + `use_flash` → 按用户速度计算 gap → 闪光曝光 |
| **SELF-TIMER** | shutter_task 跳过（不拍摄） | shutter_task 跳过（不拍摄） |

## shutter_task 曝光时序

```
shutter_task (prio 10) — 由 xTaskNotifyGive 触发，执行完后回到阻塞态

  ┌─ 1. 关闭快门 ───────────────────────────────────────┐
  │    SOL1 100% 吸合 (30ms) → 40% 保持                  │
  ├─ 2. 电机启动，反光板上升 ──────────────────────────────┤
  │    电机转 → 等 S3 变高（反光板就位）→ 电机停           │
  ├─ 3. Y Delay ─────────────────────────────────────────┤
  │    if has_flash: SOL2 吸合（光圈打开）                 │
  │    固定 18ms Y 延时                                   │
  ├─ 4. 曝光（按 mode 分支） ──────────────────────────────┤
  │    mode='1' + 无闪光: 开快门 → gap 延时 → [步骤5关]   │
  │    mode='1' + 有闪光: 开快门 → 47ms → FF→1ms → gap   │
  │    mode='B':         开快门 → 15ms → [闪光] → 等释放  │
  │    mode='T':         开快门 → [闪光] → 等释放→等再按   │
  ├─ 5. 关闭快门 ────────────────────────────────────────┤
  │    SOL1 100% 吸合 (30ms) → 40% 保持 (18ms)           │
  ├─ 6. 光圈归位 ────────────────────────────────────────┤
  │    if has_flash: SOL2 断开                           │
  ├─ 7. 电机启动吐片 ─────────────────────────────────────┤
  │    电机转 → 等 S5 变低（胶片到位）→ 电机停 → SOL1释放  │
  └─ 8. 等 S1T 释放（防连拍） ────────────────────────────┘
```

### 普通曝光 (mode='1', 无闪光)

```
sol1_release()                     // 快门打开
delay_us(shutter_delay × 100)     // 按快门速度表延时（单位 0.1ms）
↓ 进入步骤 5
sol1_pull() → 30ms → sol1_hold()  // 快门关闭
```

### 闪光灯曝光 (mode='1', 有闪光)

```
sol1_release()                     // 快门打开
delay_us(47000)                    // 47ms — 等待快门完全打开
FF_PIN = 1  →  1ms  →  FF_PIN = 0 // 触发闪光灯
delay_us(gap × 100)               // 剩余延时
↓ 进入步骤 5
sol1_pull() → 30ms → sol1_hold()  // 快门关闭
```

其中 `gap = max(shutter_delay_x10 - 470, 0)`，闪光同步时总曝光时间 = 47ms + 1ms + gap。

### B 门 + 闪光

```
sol1_release()                     // 快门打开
delay_us(15000)                    // 15ms 稳定
delay_us(32000)                    // 额外 32ms，累计 47ms 后
FF_PIN = 1  →  1ms  →  FF_PIN = 0 // 触发闪光灯
while (S1T 释放)                   // 用户控制曝光时间
↓ 进入步骤 5 关快门
```

### T 门 + 闪光

```
sol1_release()                     // 快门打开
delay_us(47000)                    // 47ms
FF_PIN = 1  →  1ms  →  FF_PIN = 0 // 触发闪光灯
while (S1T 释放)                   // 第一次按下结束
while (S1T 再次按下)               // 第二次按下结束曝光
↓ 进入步骤 5 关快门
```

## 闪光灯检测

`control_task` 主循环中每 30ms 执行一次 S2 引脚防抖检测：

```c
// 闪光灯检测（带防抖）
static int flash_debounce = 0;
bool flash_level = (gpio_get_level(S2_PIN) == 0);
if (flash_level) {
    flash_debounce = min(flash_debounce + 1, 10);
} else {
    flash_debounce = max(flash_debounce - 1, 0);
}
camera_state.has_flash = (flash_debounce >= 8);
```

- S2 拉低 = 闪光灯接入
- 防抖阈值：连续检测到 8 次（~240ms）才确认有闪光灯
- 释放阈值：连续 2 次（~60ms 未检测到）开始递减

AUTO 模式下检测到闪光灯时，`shutter_speed` 被固定为 index 13（1/30s）：

```c
if (camera_state.menu == 0) {
    camera_state.shutter_speed = camera_state.metering.auto_shutter_pos;
    if (camera_state.has_flash) {
        camera_state.shutter_speed = 13;  // 1/30s
    }
}
```

## OLED 显示布局

```
┌──────────────────────────────────────────────┐
│ SX70z          B        EV13.6 L430.12       │
│ 3s     OFF     FLASH                          │
├──────────────────────────────────────────────┤
│ 上排左: 快门速度大字    上排右: EV / LUX      │
│ 下排左: 闪光灯状态      下排中: 拍摄模式       │
│         (OFF/FLASH)          (AUTO/B/T/speed) │
└──────────────────────────────────────────────┘
```

- `has_flash=false` → 显示 `"OFF"`
- `has_flash=true`  → 显示 `"FLASH"`
- `cam_mode` 在 AUTO 模式实时反映测光结果
- `menu=3`(MANUAL) 时 `cam_mode` 显示当前选择的快门速度字符串
